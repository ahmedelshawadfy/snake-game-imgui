# Snake Game - DirectX9 Edition

A classic Snake game built with ImGui and DirectX 9, featuring a modern UI with separate control and game windows.

## Features
- 🐍 Classic snake gameplay with smooth controls
- 🎮 Arrow keys or WASD for movement
- ⏸️ Pause/Resume functionality
- 📊 Score and high score tracking
- 🎚️ Adjustable game speed (Slow, Normal, Fast)
- 🖼️ Pop-up game window for focused gameplay
- 📋 Help dialog with game instructions

## Controls
- **Arrow Keys / WASD**: Move snake
- **Space**: Pause/Resume
- **R**: Reset game
- **Mouse**: UI navigation

## How to Play
1. Click "NEW GAME" to start
2. Press any arrow key to begin
3. Guide the snake to eat the red food
4. Grow longer and avoid hitting walls or yourself
5. Earn 10 points per food eaten

## Technologies
- C++14
- DirectX 9
- ImGui (Dear ImGui)
- Windows API
